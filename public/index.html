Copy the generated SVG code to this link. Save the svg from there.


Then in your terminal, go to the folder (where you downlaoded the .svg file)
by dropping the folder in your terminal:

$ cd PATH/TO/YOUR/DOWNLAOD/FOLDER/OF/THE/SVG/FILE

Then run the following:

$ sed -i '' 's/version="1.1"/version="1.1" xmlns="http:\/\/www.w3.org\/2000\/svg"/g' yourFileName.svg

Or else it's still a valid SVG but due to lack of namespacing bug it doesn't show up properly.
The above code adds a namespace for the styling of SVG and makes it readable.
